## 🎯 Propósito do Projeto
Este projeto tem como objetivo simular o desenvolvimento de uma miniaplicação web, praticando conceitos fundamentais de controle de versão, como o uso de branches, commits, pull requests, tags e a gestão de releases.

## 🗓️ Plano de Releases Previsto
O projeto seguirá o cronograma de entrega abaixo:

* **Release v0.1:** Criação da tela de login inicial com funcionalidade de "em construção".
* **Release v0.2:** Evolução da tela de login com acesso direto à página do administrador (sem validação).
* **Release v1.0:** Implementação da lógica de autenticação completa (Admin, Operador e mensagens de erro).

## 🛠️ Tecnologias Utilizadas
* HTML5.
* Git e GitHub para controle de versão.
